Dark Mode with CSS Variables

Lee Cooper May 22nd, 2022

Why Dark Mode?

Dark mode has become an increasingly popular option for websites. The ability to toggle between light and dark themes allows users to select a presentation that can reduce eye strain, improve device battery life, and match the aesthetic that the user prefers.

In this article, you'll learn how to implement dark mode for a website using CSS Variables so that you can begin to provide this features to your users.

CSS Variables

The magic lies within CSS variables, which allows setting CSS values to a variable that can later be overwritten. This forms the basis of light and dark mode.

To start out, define a variable for the background and text colors on the root element of the document. Additionally, assign these colors to HTML elements

          
              :root {
                --background-color: #000;
                --text-color: #FFF;
              }
        
              h1, p {
                color: var(--text-color);
              }
        
              body {
                background-color: var(--background-color);
              }

In Conclusion

The CSS Box Model is a key introductory concept to the world of CSS. Once it is understood that everything on a web page is made up of boxes, it becomes easier to view how each element is sized and its behavior around other elements.
