Flexbox

Author: Lee Cooper

What is Flexbox?

Flexbox is a one-dimensional layout system in CSS that makes it easy to align and distribute content within a container, allowing for changes in screen sizes and content to be handled gracefully.

It's called "flexbox" because it allows you to make the items inside the container flexible, so they can adjust their size and position depending on the available space.

High Level Flexbox Questions

  • What: One-dimensional layout and alignment system
  • Where to use: Centering items, webpage layouts, responsive behavior, etc.
  • Why: It was designed for creating layouts and aligning content and offers a simple and intuitive approach for doing so

Flex Container and Flex Items

Flexbox requires two things: a flexbox container, and items to put in it. For example:

  • A cupboard (container) containing drink glasses (items)
  • A drawer (container) holding socks (items)
  • A wall (container) with mounted pictures (items)

<div class="container">
  <div>Flexbox Item</div>
  <div>Flexbox Item</div>
  <div>Flexbox Item</div>
</div>
.container {
  display: flex;
}

Flexbox, if available in the real world, would allow your cupboards, sock drawers, and pictures to neatly adjust to the size of their containers when they change (if they could) and react gracefully when new items are added.

Think of the organization and aesthetic benefits!

But we'll settle for the magic in webpages for now.

The Main and Cross Axis

Flexbox has two axes that defines the direction of container content:

  • Main Axis: defined by the flex-direction property (row or column)
  • Cross Axis: runs perpendicular to the main axis

The default flex-direction is "row."

This means that items, when put into a flex container, will automatically line up based on the main axis direction.

Again this is one-dimensional - it's either a row, or a column.

Once set, there's a number of properties we can apply to the container to tell it how to arrange our items, and to the items themselves to configure how much space is taken.

There's a curveball later with flex-direction and how some of these properties apply. Make sure you understand the main axis / cross axis - it'll come in handy later.

Flex-Container: Justify Content

How the browser distributes space between and around content items along the main axis of the flex container

Mental Models:
  • Aligning paragraphs in Microsoft Word (left, center, right aligned)
  • Arranging pictures on the wall (aligned with even space between)
  • Movie Theater seating (seats all centered together, or separated by aisles)

Demo üé•

.container {
  display: flex;
  justify-content: flex-start;
}

Flex-Container: Align Items

The CSS align-items property sets the align-self value on all direct children as a group. In Flexbox, it controls the alignment of items on the Cross Axis.

Mental Models:
  • How high you hang a row of photos on a wall
  • Lining up text from irregularly shaped objects

Demo üé•

.container {
  display: flex;
  align-items: stretch;
}

Flex-Container: Flex Direction

The flex-direction CSS property sets how flex items are placed in the flex container defining the main axis and the direction (normal or reversed).

Mental Models:
  • Horizontal or vertical lines
  • Rows or columns

Demo üé•

.container {
  display: flex;
  flex-direction: row;
}

Flex-Item: Flex-Grow

The flex-grow CSS property sets the flex grow factor, which specifies how much of the flex container's remaining space should be assigned to the flex item's main size.

Demo üé•

.container {
  display: flex;
}

.tile-one-flex-grow{
  flex-grow: 1;
}

.tile-two-flex-grow{
  flex-grow: 0;
}

.tile-three-flex-grow{
  flex-grow: 1;
}

Flex-Item: Flex-Shrink

The flex-shrink CSS property sets the flex shrink factor of a flex item. If the size of all flex items is larger than the flex container, items shrink to fit according to flex-shrink.

Demo üé•

.container {
  display: flex;
}

.tile-one-flex-shrink{
  flex-shrink: 1;
}

.tile-two-flex-shrink{
  flex-shrink: 0;
}

.tile-three-flex-shrink{
  flex-shrink: 1;
}
