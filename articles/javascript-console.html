Console.table()

It's no slouch!

Console.table() is a really neat visual alternative to console.log() when viewing array-like or object data.

Consider the following code:

const music = [
  { artist: "STFKR", song: "Dear Stranger", album: "Future Past Life" },
  { artist: "Foster the People", song: "The Things We Do", album: "The Things We Do" },
  { artist: "Pond", song: "Paint Me Silver", album: "The Weather" }
];

console.table(music);

Will produce the output of:

It also works for objects that are missing data, or have different properties

const music = [
  { artist: "STFKR", song: "Dear Stranger", album: "Future Past Life" },
  { artist: "Foster the People", song: "The Things We Do" },
  { song: "Paint Me Silver", album: "The Weather", releaseDate: "05/05/2017" }
];

console.table(music);
